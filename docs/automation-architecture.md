# TopShelfWebsites Automation Architecture

## Mission

Deploy owned, managed business websites within 72 hours using automation.

## Pipeline

Lead -> Supabase -> Template -> GitHub -> Cloudflare Pages -> Client Launch

## Automation Layers

### Deployment
- Repository templates
- GitHub Actions
- Cloudflare deployment
- Supabase status updates

### Content
- Client onboarding form
- Validated content fields
- Automated site assembly

### Billing
- Stripe setup payment
- Stripe recurring subscription
- Webhook status updates

### Operations
- Admin dashboard
- Site health monitoring
- Payment status controls

## Operating Principle

Build once. Deploy repeatedly. Manage centrally.
